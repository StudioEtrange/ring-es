# ring-es
Simple bash elasticsearch management

_built upon stella toolkit_


* Support ES 1.6+, ES 2.x
* Support Kibana 4.x

## Installation

	git clone https://github.com/StudioEtrange/ring-es

	cd ring-es
	./stella-link.sh bootstrap

## Usage

	cd ring-es
	./do.sh --help
